# AQI Lens

AQI Lens is a health-focused web app that helps users monitor air quality (AQI) in real time and understand pollution sources.

## Features
- Real-time AQI data
- Pollution source detection
- Clean and simple interface

## Tech Stack
- HTML, CSS, JavaScript
- AQI / Pollution data APIs
